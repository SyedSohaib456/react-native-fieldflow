# FieldFlow Monorepo

This repository contains the `react-native-fieldflow` library and an example application (`fieldflow-showcase`) demonstrating its usage.

## Structure

- `packages/fieldflow`: The core library
- `example`: An Expo application using the library

## Installation

```bash
bun install
```

## Running the Example

```bash
# Start the Metro bundler and run the app on iOS
bun run ios

# Or run on Android
bun run android

# Or run on Web
bun run web
```
